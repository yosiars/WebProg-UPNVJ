CSS adalah singkatan dari Cascading Style Sheet merupakan suatu cara untuk memformat atau membuat layout halaman web menjadi lebih menarik dan mudah dikelola. Fungsi : mendefenisikan style untuk suatu teks dengan jenis huruf, ukuran, warna tertentu. Di dalamnya terdapat banyak style. CSS dapat dituliskan pada bagian suatu dokumen HTML atau diletakkan di sebuah file eksternal. Perintah-perintah CSS dibatasi oleh tag Keuntungan Menggunakan CSS adalah Ukuran file lebih kecil, Load file lebih cepat, Dapat berkolaborasi dengan JavaScript, Pasangan setia XHTML, Menghemat pekerjaan tentunya, dimana hanya membuat 1 halaman CSS, Mudah mengganti-ganti tampilan dengan hanya merubah file CSS nya saja Kekurangan dari CSS menurut saya adalah tampilan pada browser berbeda-beda, Terdapat browser yang tidak support CSS (browser lama), harus tahu cara menggunakannya, dibutuhkan waktu lebih lama dalam membuatnya, ada kemungkinan bug/error dalam CSS.
